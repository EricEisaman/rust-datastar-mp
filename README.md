# RAD Baby

____

![RAD Baby](https://raw.githubusercontent.com/EricEisaman/assets/main/images/RAD_Baby.jpg)

A super simple 2d physics server-administered multiplayer demo using the **RAD Baby** stack, comprised of <a href="https://doc.rust-lang.org/book/" target="_none">Rust</a> <a href="https://docs.rs/axum/latest/axum/" target="_none">Axum</a> <a href="https://data-star.dev/" target="_none">Datastar</a> and <a href="https://www.babylonjs.com/" target="_none">Babylon.js</a>.
